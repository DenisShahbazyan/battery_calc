# Расчёт идеального веса АКБ по ёмкости

Идеальный вес свинцово-кислотного аккумулятора напрямую зависит от его ёмкости. Базовое соотношение: на каждые **10 Ач** приходится **2,8 кг** массы.

## Формула расчёта

> **Вес АКБ (кг) = 0,28 × Ёмкость АКБ (Ач)**

где **0,28** — коэффициент удельного веса (кг/Ач).


> _Примечание: указанные значения являются идеальными расчётными. Фактический вес АКБ может отличаться в зависимости от производителя, типа электролита и конструкции корпуса._
